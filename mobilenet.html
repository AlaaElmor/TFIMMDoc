MobileNet

MobileNet is a convolutional neural network architecture that seeks to perform well on mobile devices.it is based on a streamlined architecture that uses depth-wise separable convolutions to build light weight deep neural networks.

Versions
MobileNetV1
MobileNetV2



MobileNetV1

To load a MobileNetV1 Model:

from .MobileNet import MobileNetV1
from keras.layers import Input

input_layer = Input(shape=(h,w,3,))
model=MobileNetV1(num_classes=10)(input_layer)



MobileNetV2 The intermediate expansion layer uses lightweight depthwise convolutions to filter features as a source of non-linearity. As a whole, the architecture of MobileNetV2 contains the initial fully convolution layer with 32 filters, followed by 19 residual bottleneck layers.

To load a MobileNetV2 Model:

from .MobileNet import MobileNetV2
from keras.layers import Input

input_layer = Input(shape=(h,w,3,))
model=MobileNetV2(num_classes=10)(input_layer)
