AblationCAM

Ablation-based Class Activation Mapping (Ablation CAM) uses ablation analysis to determine the importance (weights) of individual feature map units w.r.t. class. Further, this is used to produce a coarse localization map highlighting the important regions in the image for predicting the concept. Our objective and subjective evaluations show that this gradient-free approach works better than state-of-the-art Grad-CAM technique. How do I use this technique on an image?

To load a AblationCAM class:

The constractor have three parameters ( Model ) , (layer-name) and (Class-index)

We will use Resnet50 model for test

from tensorcam import AblationCAM
from tensorflow.keras.applications import ResNet50

class_index = {'tiger_cat': 282, 'beagle': 162}
cam = AblationCAM(model, class_index['tiger_cat'] ,layer_name='conv5_block3_out')
The AblationCAM class have 3 methods :

(get_image) : that takes the original image and retern the image as a numpy 2d array and the origenal image self.
(compute_cam_features) : that takes a 2d numpy array and return the heatmap.
(plot_images) : that takes the original image , the heatmap and return the image with the heatmap.

image, orig = AblationCAM.get_image('sample.jpg')
cam_features = cam.compute_cam_features(image)
AblationCAM.plot_images(orig, cam_features)
